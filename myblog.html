Starting DevCode Camp

Learning Javascript

Today 01-01-16 we dove into Javascript head-on, I struggled with the whole idea of functions, and I found it very hard to grasp, but once I completed my project, a lot more became clear. I've found that I just need to ask questions and use trial and error to my advantage. I've also learned that I need to make a copy before I start doing trial and error, and I also need to make my GitHub Repo. Eventhough this class is overwhelming at the very least, but I'm excited to see what else I'll be learning in DevCode.

Dice Game Project

Today 01-07-16 we have completed our first Javascript game! We worked on it in a group, the group decided to call the game "Walking Simulator 2016", and that's exactly what it is. The goal of the game is to roll a dice (which we wrote out the code for) and to pass to the next level you need to roll a certain number. I feel that I learned much more because I worked with a group which helped me develop better interpersonal skills. Completing this game has also given me a much better undertanding of the building blocks of code. Recently I've learned something that I will keep with me forever...
